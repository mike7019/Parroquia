9️⃣  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

❌ "No se puede conectar desde iPhone"
   ✅ Verificar firewall: sudo ufw allow 3000/tcp
   ✅ Reiniciar Docker: docker-compose down && up -d
   ✅ Verificar DNS: ping 206.62.139.100

❌ "Error CORS en login"
   ✅ Limpiar caché: Settings → Safari → Clear History
   ✅ Reiniciar Safari en iPhone
   ✅ Esperar 1 minuto después del deploy

❌ "Descarga lenta"
   ✅ Usar WiFi en lugar de datos móviles
   ✅ Cerrar otras apps
   ✅ El servidor limita a 50 items por defecto en móvil

❌ "No aparecen datos"
   ✅ Verificar headers: curl -i http://...
   ✅ Revisar logs: docker-compose logs -f api
   ✅ Recargar página: Pull down en Safari
